Programación Estructurada

La programación estructurada se basa en el uso de estructuras de control como secuencias, condicionales y bucles.

Variables

Las variables son contenedores que almacenan datos. Pueden ser números, cadenas de texto, booleanos, etc.


let nombre = "Juan";
let edad = 25;
console.log(nombre, edad);

Bucles

Los bucles permiten repetir acciones múltiples veces.


for (let i = 1; i <= 5; i++) {
    console.log("Iteración:", i);
}

Funciones

Las funciones agrupan código reutilizable.


function saludar(nombre) {
    return `Hola, ${nombre}!`;
}
console.log(saludar("Juan"));
