Programación Estructurada

La programación estructurada se basa en el uso de estructuras de control como secuencias, condicionales y bucles.

Conceptos Básicos

  • Variables: Almacenan datos.
  • Condicionales: Permiten tomar decisiones (if, else).
  • Bucles: Repiten acciones (for, while).
  • Funciones: Agrupan código reutilizable.

Ejemplo Práctico

Este ejemplo muestra cómo usar una función para calcular el factorial de un número:


function factorial(n) {
    if (n === 0 || n === 1) return 1;
    return n * factorial(n - 1);
}
console.log(factorial(5)); // Salida: 120
